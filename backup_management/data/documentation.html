Backup Management User Guide

Automated Volume Synchronization

The system automatically synchronizes backup volumes for both Kopia and pgBackRest engines. This ensures that the local database is aware of all available snapshots and their statuses.

[@ANCHOR: UX_BACKUP_SYNC]

Security and Multi-Tenancy

All backup operations are performed with minimum privilege using dedicated service accounts. Multi-website and multi-company environments are fully supported, ensuring data isolation between different tenants.

Restore Operations

Administrators can trigger restores of specific snapshots via the Restore Wizard. The system validates target paths to prevent unauthorized access or accidental overwrites of critical system files.
