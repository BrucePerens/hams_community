User Websites SEO Guide

This module extends the user_websites functionality by integrating Odoo's native SEO tools into personal and group websites.

Overview

User Websites SEO allows users to optimize their blogs for search engines. It adds SEO metadata fields to user profiles and group websites, enabling the "Optimize SEO" widget on the frontend.

Features

  • SEO Metadata: Title, Description, Keywords, and Open Graph Image for personal blogs.
  • Frontend Integration: Users can edit SEO metadata directly from their blog pages using the standard Odoo SEO tool.
  • Security: Follows the zero-sudo architecture, allowing users to edit only their own metadata through a dedicated service account.

How to Use

  1. Navigate to your personal or group blog on the website.
  2. If you are the owner or a member of the group, you will see the "Site" menu.
  3. Select "Optimize SEO" from the menu.
  4. Fill in the metadata and save.

Developer Information

This module inherits website.seo.metadata into res.users and user.websites.group. It whitelists these fields in res.users for self-writing and overrides the write method to escalate privileges via the user_websites.user_user_websites_service_account service account when necessary.
