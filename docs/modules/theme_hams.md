# 🎨 Hams.com Theme (`theme_hams`)

*Copyright © Bruce Perens K6BP. AGPL-3.0.*

**Context:** Technical documentation strictly for LLMs and Integrators.

## 1. Overview
A specialized frontend theme heavily modifying Bootstrap 5 to produce a rugged, hardware-focused UI.

## 2. Design Elements
* **Morse Code Borders:** SVG backgrounds used as separators spelling `WELCOME TO HAMS.COM`.
* **Digital Displays:** Includes a `.digital-display` utility class and dynamic "Last QSO" OWL snippets.
* **Hardware Inputs:** Features interactive telegraph key SVG components and tactile push-button active states (`translateY(4px)`).
