## 1. Overview
A dedicated Odoo 19 theme module that applies a kitschy, ham-radio-inspired aesthetic to the platform.

## 2. Design Elements
* **Morse Code Borders:** SVG-powered borders spelling out `WELCOME TO HAMS.COM`.
* **Hardware Palette:** Olive Drab, Amber Glow, Matte Black.
* **Digital Displays:** `.digital-display` utility class. Includes a dynamic "Last QSO" widget (`s_digital_display.js`) pulling from the Logbook API.
* **Telegraph Key:** Interactive SVG Morse Code key (`s_morse_key.js`).
